Object.observe

Basic Concept


  • Native support for MVC
    • Declares a callback to observe changes of an object
      (e.g. C.U.D. of the object's properties) 
    
var model = {};
Object.observe(model, function (changes) {
    changes.forEach(function (c) {
        alert(' type: ' + c.type +
              ' name: ' + c.name +
              ' old value: ' + c.oldValue);
    });
});
  • Spec: ECMAScript 7
  • Support: Chrome 36+, Node harmony
    (ref. ECMAScript 7 compatibility)

Object.observe

Custom Notification 



var rect = new Rectangle();

Object.observe(rect, function (changes) {
    changes.forEach(function (c) {
        if (c.name === 'area') {
            alert('Area changed to ' + c.object.area + ' by ' + c.by);
        }
    });
});

function Rectangle() {
    var _width, _height; // hidden store for width and height

    var notifier = Object.getNotifier(this);

    Object.defineProperty(this, 'area', {
        get: function () { return _width * _height; }
    });
    Object.defineProperty(this, 'width', {
        get: function () { return _width; },
        set: function (arg) {
            notifier.notify({
                type: 'update',
                name: 'area',
                by: 'width'
            });

            _width = arg;
        },
    });
    Object.defineProperty(this, 'height', {
        get: function () { return _height; },
        set: function (arg) {
            notifier.notify({
                type: 'update',
                name: 'area',
                by: 'height'
            });

            _height = arg;
        }
    });

    this.width = this.height = 1;
}

Web Components

Basic Concept


  • Define new HTML elements as reusable DOM components
  • Consists of ...
    • Custom Elements: Register new HTML element names
    • HTML Templates: Define presentation layers of reusable components
    • Shadow DOM: Glue templates to custom elements with encapsulation
    • HTML Imports: Single definition and multiple use
  • Spec: WhatWG, W3C
  • Support: Chrome 36+, Opera 25+, Android 37+, Chrome for Android 38+ (ref. Can-I-Use)

Custom Elements @ Web Components

Basic Concept


  • Register a new HTML element name (tag name) to a browser using document.registerElement() 
    
var NameTag = document.registerElement('name-tag');
document.body.appendChild(new NameTag());
    • Element names must have a dash for forward compatibility
    • document.registerElement(...) returns the constructor of the new element

Custom Elements @ Web Components

Extending Another Element


  • Can extend other element 
    
var ExtendedButton = document.registerElement('extended-button', {

HTML Templates @ Web Components

Basic Concept


  • A new element <template> with special properties
    • Hidden: Parsed but not rendered
    • Inactivated: Scripts don't run, images don't load, audio doesn't play, etc
  • How to use a template
    • Clone its content (tmpl.content, a DocumentFragment object)
      and add to the DOM 
    
var tmpl = document.querySelector('#mytemplate');
var clone = document.importNode(tmpl.content, true); // deep clone
document.body.appendChild(clone);

Shadow DOM @ Web Components

Basic Concept


  • Provides encapsulation for reusable components
    • Separates the presentation detail from semantics and hides it
    • Avoids name collision: IDs, class names, styling rules, etc
  • A simple example
    • Add a shadow root to an element (shadow host) using createShadowRoot method
    • If an element has a shadow root, the shadow root's content is rendered instead
